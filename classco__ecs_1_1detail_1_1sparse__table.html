Detailed Description

template<typename K, typename T, bool is_map, typename Allocator>
class co_ecs::detail::sparse_table< K, T, is_map, Allocator >

Sparse table implementation.

The underlying layout is: dense: |value_type|value_type|value_type|value_type| sparse: |key_type | |key_type |key_type |

Template Parameters
KKey type
TMapped type type
is_mapWhether this is for a map or set
